菜单项 #

BmContextMenuItem 菜单项是右键菜单的子组件，请勿在其他组件中使用。

属性 #

属性名类型默认值描述
callbackFunctionnew Function点击菜单时执行的回调函数，第一个参数为 {BMap, map, target, pixel, point}
textString指定此菜单项的文本
iconUrlString指定此菜单项的icon URL（大小为17px*17px）
disabledBooleanfalse是否禁用菜单项
seperatorBooleanfalse是否是分隔线（此属性为真时，其它属性会被忽略）
