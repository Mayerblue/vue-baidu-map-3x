自定义控件 #

BmControl 组件允许开发者自由定制控件。

由于受到百度地图的鼠标滚轮缩放机制影响，在自定义控件组件中使用鼠标滚轮操作一个 DOM 元素的滚动条会失效，该问题可以通过阻止自定义控件的 wheel 事件冒泡来解决。

属性 #

属性名类型默认值描述
anchorControlAnchor控件停靠位置，默认为左上。
offsetSize控件偏移值
