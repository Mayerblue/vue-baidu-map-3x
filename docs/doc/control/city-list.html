城市选择控件 #

BmCityList 城市列表控件仅在地图视图尺寸足够大时生效。

属性 #

属性名类型默认值描述
anchorControlAnchor控件停靠位置
offsetSize控件偏移值

事件 #

事件名参数描述
changeBefore切换城市前触发此事件
changeAfter切换城市后触发此事件
