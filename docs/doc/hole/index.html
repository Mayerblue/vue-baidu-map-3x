注意事项 #

WARNING

地图初始须设置center，zoom否则地图不会初始化， 地图须设置高度才能展示

<baidu-map class="map" :center="{lng: 116.404, lat: 39.915}" :zoom="15">
</baidu-map>

<style>
.map {
  width: 100%;
  height: 300px;
}
</style>

	</div>
